# 🎗️ CancerSense — Breast Cancer Predictor

A machine learning web application that predicts whether a breast tumor is **Benign** or **Malignant** based on diagnostic measurements. Built with Flask, Scikit-learn, and vanilla HTML/CSS/JS.

---

## 🖥️ Demo

| Prediction Page | Model Comparison Page |
|---|---|
| Enter 30 diagnostic features | Compare Logistic Regression, Decision Tree & Random Forest |

---

## 🧠 Models Used

| Model | Description |
|---|---|
| Logistic Regression | Fast, interpretable linear classifier |
| Decision Tree | Rule-based non-linear classifier |
| Random Forest | Ensemble of 100 decision trees |

All models are trained on the **Wisconsin Breast Cancer Dataset** (569 samples, 30 features) from `sklearn.datasets
